# TransitOS

  Plataforma de gestión de operaciones de transporte. Full-stack monorepo con React + Vite (frontend) y Express 5 + PostgreSQL (backend).

  ## Stack
  - **Frontend:** React 18, Vite, TypeScript, TailwindCSS, Recharts, Framer Motion
  - **Backend:** Express 5, Drizzle ORM, PostgreSQL, JWT, bcryptjs
  - **Monorepo:** pnpm workspaces

  ## Roles
  - **Admin:** gestión completa de viajes, usuarios, WhatsApp y dashboard
  - **Conductor:** vista de sus viajes, actualización de estado en tiempo real
